[directcopula_simul.r]
- main programme to generate sample data and estiamte the copula models proposed in Kim et al. (Copula-based Direct Utility Models for Correlated Choice Alternatives, QME, 2022)
- it can simulate four models (benchmark, power, sine, Gaussian) 

[directcopula_functions_cpp]
- functions needed to run the main code written in CPP

[directcopula_functions_r]
- functions needed to run the main code written in R
